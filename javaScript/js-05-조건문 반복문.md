# javaScript 조건문 반복문

<br />

### - if문

  - 조건에 따라 switch문으로 바꿀 수 없을 수 있다.

  종류 | 문법
  --|--
  if문 | if() {}
  if else문 | if() else {}
  else if문 | else if() {}

<br />

### - switch문 : 
  - 조건이 어떤 값으로 딱 떨어질 때
  - 언제나 if문으로 바꿀 수 있다.

  종류 | 문법
  --|--
  switch문 | switch()
  switch case braek문 | switch() { case "값": braek }
  switch default문 | switch() { default: }

<br />

### - for문 

  종류 | 문법
  --|--
  반복문 | for(시작조건; 종료조건; 변화조건) {}
