# Vue 설치방법

<br />

## - CDN
  - 프로토 타이핑 또는 학습목적

## - NPM
  - 대규모 애플리케이션을 구축할 때 권장하는 방식

## - CLI
  - 단일 페이지 애플리케이션을 빠르게 구축하는 방식
    - 장점 : 구성 옵션을 최대한 신경 쓰지 않고 문법에만 집중해서 프로젝트를 만들 수 있다.
    - 단점 : 세부적인 옵션들을 정리하면서 프로젝트를 관리하기는 어렵다.

> [설치 링크](https://kr.vuejs.org/v2/guide/installation.html)
